### Exemplo de definição de envelopamento de tags de hierarquia :

```html

<body>
  <header>
    <nav> </nav>
  </header>
<section>
   <aside> </aside>
   <article> </article>
</section>

<footer> </footer>

</body>
```
### Definição apropriada de cada tag :
```html

<body>

  <!-- corpo do site  -->

</body>

```

```html

<header>

  <!-- cabeçalho do site  -->

</header>

```

```html

<nav>

  <!-- menu de navegação do site -->

</nav>

```


```Html

   <!-- seção do site -->

</section>

```


