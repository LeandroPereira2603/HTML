## Conheçendo todas as Tags

 <p> Tags para formatação de texto  </p>

```html 

<p> 

 <!-- Formata o texto com parágrafo--> 
 
</p> 


```

```html 

<strong>

 <!-- Formata o texto em negrito --> 

</strong >

```

```html 

<a>

   <!-- Formatar o texto para o acesso de link -->

</a>

```

```html 

<abbr>

   <!-- Formata palavra em uma sigla abreviada -->

</abbr>

```

```html
</table>

```

```html

<div>

</div>


```

```html

<audio>
```
